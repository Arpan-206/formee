# Installing the CLI

1. Install *formee* with pip

```bash
pip3 install formee
```

2. Use the Github repository

    ```bash
    git clone https://github.com/Arpan-206/formee.git
    cd formee
    ```

    - If you use poetry, then:
        ```bash
        poetry install
        ```
    - Otherwise, use pip
        ```bash
        pip3 install -r requirements.txt
        ```
